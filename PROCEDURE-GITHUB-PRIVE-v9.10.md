# Procédure GitHub privé

1. Créer un dépôt GitHub privé.
2. Copier le contenu du ZIP v9.10 dans un dossier propre.
3. Vérifier qu’aucun `.env` réel n’est présent.
4. Initialiser Git : `git init`.
5. Ajouter les fichiers : `git add .`.
6. Contrôler : `git status`.
7. Commit : `git commit -m "DL Creator Web v9.10 github security hardening"`.
8. Ajouter le remote privé.
9. Push sur la branche principale.
