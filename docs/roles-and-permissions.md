# Rôles et permissions

## Rôles cibles

- `admin`
- `chefFormation`
- `responsableDomaine`
- `redacteur`
- `validateur`
- `lecteur`

## Compatibilité locale

Les anciens droits locaux restent supportés :

- `ADMIN STRUCTURE APPLICATION` → `admin`
- `GESTION DL` → `chefFormation`
- `RÉDACTION DL` → `redacteur`
- `CONSULTATION DL` → `lecteur`

## Actions préparées

- création DL
- lecture DL
- modification DL
- suppression DL
- validation
- publication
- audit
- gestion utilisateurs
- backup
