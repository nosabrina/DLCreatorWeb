# Templates e-mails transactionnels — v8.90

Templates préparés et versionnés pour activation future via Netlify Functions ou backend Node : invitation utilisateur, reset password, validation demandée/refusée, publication validée/refusée, notification admin et rappel validation.

Aucun secret ne doit être ajouté au frontend. Les providers Postmark, SendGrid et Mailgun restent désactivés par défaut.
