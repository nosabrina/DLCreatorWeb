# Procédure Netlify Preview

1. Connecter Netlify au dépôt GitHub privé.
2. Build command : vide.
3. Publish directory : `.`.
4. Functions directory : `netlify/functions`.
5. Définir uniquement des variables désactivées ou placeholders.
6. Lancer un Deploy Preview.
7. Vérifier le chargement, diagnostic, audit local et fonctions préparées.
